# Collabio | Online whiteboard

Real-time whiteboard made with Next.JS and Socket.IO
## Features

- Drawing lines, circles and rectangles
- Eraser
- Undo/Redo
- Real-time mouse tracking
- Saving canvas
- Changing backgrounds
- Sharing
## Made using
- Next.JS
- Recoil
- TailwindCSS
- Framer Motion
- Socket.IO

## Installation
1. Clone Repo
2. run `npm install`
3. run `npm run dev`
4. Open `http://localhost:3000`
